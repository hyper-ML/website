    Download

    Hyperflow requires two components to make your model building cycle more effective.

    hyperflow Command Line (CLI)

    hflow is a client side component ****that every user will need to install on her personal machine.

    hyperflow Server

    hflserver can be installed on a cloud machine or on-premise. It does most of the hand shaking between your system and server resources. It acts as a broker between with CLI and workers. And also handles version control.

    Install hyperflow Command Line (CLI)

    OSX

    curl -fsSL -o /usr/local/bin/hflow https://github.com/hyperflow-ai/hyperflow/releases/download/0.0.1/hflow-osx-10.13.6
chmod +x /usr/local/bin/hflow

    Install hyperflow Server

    OSX

    curl -fsSL -o /usr/local/bin/hflserver https://github.com/hyperflow-ai/hyperflow/releases/download/0.0.1/hflserver-osx-10.13.6
chmod +x /usr/local/bin/hflserver
